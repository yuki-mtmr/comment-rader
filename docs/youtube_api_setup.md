# YouTube Data API v3 有効化 & APIキー取得手順

CommentRadarでYouTubeのコメントを取得するために必要な「YouTube Data API v3」のAPIキーを取得する手順です。

## 手順

### 1. Google Cloud Console プロジェクトの作成
1.  [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、Googleアカウントでログインします。
2.  画面左上のプロジェクト選択ボックス（ロゴの横）をクリックします。
3.  **「新しいプロジェクト (New Project)」** をクリックします。
4.  プロジェクト名（例: `comment-radar`）を入力し、**「作成」** を押します。
5.  作成されるまで少し待ち、作成完了の通知が出たらそのプロジェクトを選択します。

### 2. YouTube Data API v3 の有効化
1.  左側のメニューから **「APIとサービス」 > 「ライブラリ」** を選択します。
2.  検索バーに `YouTube Data API v3` と入力して検索します。
3.  検索結果から **「YouTube Data API v3」** をクリックします。
4.  **「有効にする (Enable)」** ボタンをクリックします。

### 3. APIキー (Credentials) の作成
1.  APIが有効化されると、概要画面に移動します（または左メニューの「APIとサービス」>「認証情報」）。
2.  ページ上部の **「認証情報を作成 (Create Credentials)」** をクリックし、**「APIキー」** を選択します。
3.  `AIza...` で始まるAPIキーが生成されます。**このキーをコピーして、メモ帳などに一時保存してください。**

### 4. (推奨/任意) APIキーの制限
不正利用を防ぐため、キーに制限をかけることを強く推奨します。

1.  作成されたAPIキーの「キーを制限」をクリック（または認証情報一覧からキーの名前をクリック）。
2.  **「APIの制限」** セクションで **「キーを制限」** を選択。
3.  ドロップダウンから **「YouTube Data API v3」** *のみ* にチェックを入れます。
4.  **「保存」** をクリックします。
    *   *注: Webアプリとして公開する場合は「アプリケーションの制限」で「HTTPリファラー」を設定しますが、ローカル開発中は設定しなくてOKです（設定するとlocalhostから動かなくなることがあります）。*

## 次のステップ
取得したAPIキーは、プロジェクトセットアップ後に作成する `.env.local` ファイルに記述します。

```bash
NEXT_PUBLIC_YOUTUBE_API_KEY=AIzaSy...
```

準備ができたら教えてください！プロジェクトのセットアップを進めます。
