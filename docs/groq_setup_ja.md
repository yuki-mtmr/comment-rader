# Groq セットアップガイド

Groqは、非常に高速な推論が可能なLLM提供サービスです。無料枠が提供されており、Geminiのクォータ制限時の代替手段として最適です。

## 1. APIキーの取得
1. [Groq Console](https://console.groq.com/keys) にアクセスします。
2. GoogleアカウントまたはGitHubアカウント等でログインします。
3. **"Create API Key"** をクリックし、名前を付けてキーを生成します。
4. 生成されたキー（`gsk_...`）をコピーします。

## 2. 環境変数の設定
`.env.local` ファイルを開き、取得したキーを貼り付けます。

```env
GROQ_API_KEY=gsk_あなたのAPIキー
```

また、使用するエンジンをGeminiからGroqに切り替えるには、以下のように設定します。

```env
LLM_ENGINE=groq
```

## 3. 動作確認
以下のコマンドを実行して、Groqが正しく動作するか確認できます。

```bash
node test-groq.mjs
```

## 無料枠の制限 (2026年時点の目安)
- **モデル**: `llama-3.1-8b-instant` (デフォルト設定)
- **リクエスト数**: 1分間に20回まで (20 RPM)
- **1日の上限**: 14,400回 (14,400 RPD)

Gemini (15 RPM) よりもわずかにリクエスト制限が緩く、何より応答速度が非常に速いのが特徴です。
